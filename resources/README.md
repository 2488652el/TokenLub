# Resources

This folder holds notes for binary assets. Electron Builder reads static build
resources from the top-level `build/` directory.

## Required before `npm run dist:win`

- `build/icon.ico` — TokenLub app icon used by the installer and Windows taskbar.
- `build/icon.png` — TokenLub app icon used by the local dev window.

- `tray.png` — optional 16/32px tray icon used in Phase K's tray menu.

## Design

The TokenLub mark is derived from the approved green/cyan logo artwork.
