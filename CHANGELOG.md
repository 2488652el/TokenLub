# 更新日志

## Unreleased

## 1.1.2 - 2026-07-20

- API Keys 与余额查询卡片新增可拖动排序，提供抬升、目标占位、邻卡让位和落位反馈；顺序按页面持久化，并支持筛选场景、方向键操作与 reduced motion。
- 完善模型价格匹配，对大小写、供应商前缀及 Kimi K3 别名进行规范化，避免已有目录价格因模型命名差异而无法命中。
- 项目用量统一在聚合前将多币种成本折算为人民币，避免不同币种直接相加导致费用统计失真。

## 1.1.1 - 2026-07-20

- 为全部 10 个桌面端页面加入与现有设计风格一致的语义化动效，覆盖数字、图表、进度、卡片、表格、状态和弹窗反馈。
- 新增统一 Motion tokens、数字递增、进度绘制和错峰分组组件，并限制位移、时长和批量动画数量，保持界面克制稳定。
- 完善 `prefers-reduced-motion` 降级、弹窗 ESC 关闭与焦点恢复，并新增真实 Electron 路由、动画收敛和可访问性测试。

## 1.1.0 - 2026-07-18

- 重构请求日志筛选区，使用供应商下拉、中文来源切换、日期范围和模型搜索统一定位日志，并移除重复的日期跳转操作。
- 优化请求日志表格，统一中文表头、供应商名称和来源文案，将币种合并到费用展示，并改善列宽、对齐与分页信息。
- Web 控制台新增账号与设备绑定记忆、登录会话恢复和显式退出；刷新页面后可安全续期会话，退出时同步撤销服务端会话。
- 完善控制台密码变更流程：改密成功后清除本地会话并要求重新登录，同时确保密码不会写入浏览器存储。

## 1.0.9 - 2026-07-18

- 重构 API Keys 卡片，以供应商品牌色和双列布局区分 API 余额、Coding Plan、Token 资源包、组织用量与聚合网关，并补充凭据类型、来源、Key 末位和查询状态。
- 升级 ChatGPT 订阅额度卡片，集中展示当前计划、登录状态、下次重置时间，以及 5 小时和周额度的进度与已用比例。
- 模型价格表新增人民币/美元统一显示切换，自动按每条价格的原始币种获取汇率并完成高精度折算，同时保留原始币种便于核对。
- 完善多币种汇率校验与异常提示；汇率不可用时不再展示误导性金额，并补充价格折算单元测试。

## 1.0.8 - 2026-07-17

- 简化项目用量热力图，改为按日期汇总的紧凑网格，并新增按项目着色的 Token 趋势曲线和入场动画。
- 重构余额查询卡片，明确区分 API 余额、Coding Plan、Token 资源包、组织用量、聚合网关和手动额度。
- 为 Kimi、MiniMax、LongCat、DeepSeek、GLM 等供应商增加差异化配色、核心指标、进度条和无数据说明。

## 1.0.7 - 2026-07-17

- 新增“异常价格变动需审批”开关，默认开启并持久化用户选择。
- 关闭审批后，待审批价格变更立即放行，后续官方目录同步也会直接应用异常价格变化。
- 手动同步官方价格会根据审批开关决定是否需要确认。

## 1.0.6 - 2026-07-17

- 修复本机 CLI Session 解析触发逻辑：打开 API Keys 页面不再自动解析。
- 自动解析仅在开关开启时由后台调度；手动解析仅通过“解析全部”或来源卡片的“解析入库”执行。
- Dashboard、Provider 汇总和请求日志的刷新操作不再隐式扫描 CLI 日志，关闭自动解析后状态也会持续保持。

## 1.0.5 - 2026-07-16

- 新增应用内检查更新与后台自动更新：从 GitHub Releases 获取当前系统对应版本，显示下载进度，下载完成后静默覆盖安装并重启。
- 更新只替换应用安装目录，SQLite 数据库与本地设置继续保存在 Electron 用户数据目录中。

## 1.0.4 - 2026-07-16

- 云端同步设置支持选择本地备份目录，预览会显示实际备份位置。
- 新增 Ubuntu 一键部署脚本与 Docker Compose 配置，支持 HTTPS 域名和 SSH 隧道两种同步服务模式。
- 增加服务器健康检查、备份、升级和卸载命令，并补充部署验收文档与自动化测试。
- 中文 README 作为默认项目首页，加入产品截图、功能展示和 GitHub Release 下载入口。

## 1.0.3 - 2026-07-15

- Web 控制台新增一键绑定桌面应用，通过 `tokenlub://` 唤起 TokenLub。
- 绑定过程使用两分钟有效且只能兑换一次的票据，不在链接中传递密码或会话令牌。
- 桌面端支持冷启动和已运行状态下接收绑定链接，并自动保存加密同步会话。

## 1.0.2 - 2026-07-14

- 加入内置账号登录、设备与会话管理，以及无域名的 Ubuntu/Docker + SSH 隧道自托管部署方案。
- 补齐 PostgreSQL 迁移、历史清理、备份恢复、限流、隐私审计和双设备同步验证。

## 0.1.4 - 2026-07-09

- 在 0.1.3 的用量统计 UI 重构基础上，补齐请求日志分页能力，支持按页浏览与按日期跳转到当日请求列表。
